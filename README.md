# City-Subway-Service-System（城市地铁服务系统）
## 项目基础信息
- 地理信息服务作业3 11月10日版

## 问题
### 一、待解决问题
1. 登录功能：目前仅支持1个测试用户，无新用户创建入口；
2. 导航跳转逻辑：点击「用户」入口时，无论登录/未登录状态均跳转至登录页，需优化为「登录状态跳转至个人中心，未登录状态跳转至登录页」；
3. 首页状态切换：右上角「登录按钮」未实现状态联动（登录/未登录状态下按钮样式/跳转目标未切换）；
4. 路线总览页：地图线路无注记文字，点击特定线路时无高亮显示效果；
5. 站点周边页：POI（兴趣点）数据覆盖不全,风格与前2页不太一致；
6. 路线规划模块尚未开发；
7. 地图配置：Mapbox底图注记为英文，需修改为中文注记。

### 二、已修复
1. 线路详情页：筛选多条线路时，点击切换方向会出现功能异常（已完成修改）；
2. 数据格式：JSON数据坐标系错误（已修复），但线要素存在简化问题待优化。
